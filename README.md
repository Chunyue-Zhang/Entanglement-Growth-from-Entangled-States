# Entanglement Growth from Entangled States

[![arXiv](https://img.shields.io/badge/arXiv-2510.08344-b31b1b.svg)](https://arxiv.org/abs/2510.08344)

This repository provides the source code for the paper: **"Entanglement Growth from Entangled States: A Unified Perspective on Entanglement Generation and Transport"** by Chun-Yue Zhang, Zi-Xiang Li, and Shi-Xin Zhang.

The script `entanglement_dynamics.py` is used to produce the main numerical results presented in **Figure 1(d,e)** and **Figure 2** of the paper. It uses exact diagonalization to simulate the entanglement dynamics in various quantum many-body dynamics.

---

This program uses the QuSpin library. The output file contains the parameter settings, the random configurations for each sample, and the calculated data.
